# 你好——Welcome to SCU-Deep Learning生存手册

## 序言

今天是2025年12月28日，已经是我在四川大学待下来的第四个冬天了。回望本科的经历，有太多的事情值得回味，也有太多的经历值得去反思和成长。我建立这个微信交流群的想法在学期初就已经萌生，而且其实学校内部其实是有类似的相关群聊的，比如人工智能社团，或者是一些零零散散七七八八的交流群，这些群聊或疏于管理，或由于老师和领导的加入使得同学的发言显得局促。我们四川大学似乎没有一个像样的，正经的，开源的，善于经营的深度学习交流社区。而我的初衷，<u>就是建立在没有任何金钱利益的基础上，开发一个属于我们SCU自己的交流大本营</u>，在这里大家可以畅所欲言，而不善于公共聊天的同学也可以通过定期的活动从中受益。这个手册的愿望就是希望能够有数以百计的四川大学对深度学习感兴趣的同学能够<u>少走95%的弯路</u>，正式踏入该领域学习门道。同时我还希望除了纯粹的学习之外，更重要的是营造开源，友善的交流氛围，让思想在聊天的火花中碰撞，让跨领域的AI合作，成为交流群的常态。“**海纳百川，有容乃大**”，四川大学的校训始终提醒着我，要肩负起开放共享的使命，为更多的小白和学子提供广阔的交流平台。

仅靠我个人的力量是不够的，我自己也并非科班学生，只是在相关经历上或多或少比大家丰富一些，而在学习的道路上都是人人平等的。我也通过自己的“人脉”邀请了海内外诸多高校的本科优秀学生为大家答疑（shuiqun）。**<u>在此充分感谢香港大学、香港理工大学、浙江大学、武汉大学、华中科技大学、温州肯恩大学、西交利物浦大学、西湖大学等高校同学的支持。</u>**我也深切的希望能够有更多的同学参与交流群的管理，践行大本营的初衷。

在AI、大模型盛行的年代，填鸭式的悲壮的学习方式即将被淘汰，学生时代的思想在AI方向的科研和项目工作中也将会逐步被取代。**工程能力**，成为了硕博学长学姐，以及老师更看重的能力，这一点无论是工作，还是日后的升学研究都必不可少。我想在序言的开头告诉大家这个真相，是因为我已经看见太多的人因为舍不得放弃已经积累的资本，反而误入歧途。**当机立断，培养工程直觉，先做再去想，学会使用AI**，成为了快时代科研和工作的母题。

此外，我更希望，大家能够在学习和科研中，真正的了解和认识自己。人生并不应该被保研、绩点、综测而束缚，而是应该在学习的过程中找到自己所爱，哪怕你在学习中发现自己并不适合该领域，打算退学去开咖啡店，我也是佩服和尊重的。请你在看完这段序言过后好好的问问自己，我到底需要什么？同学们的青春无限，我再此送给大家最后一句话，也是我最喜欢的一句话——**着力即差**。大胆的去做吧！

## 问题集（第一期）

我觉得比起在群里回答，收集问题集进行集中的回答将会更有效率，且更能帮助到同学。在这里我会匿名上传一些同学们的相关问题，在这里进行解答（低质量或者太宽泛的问题恕我无法回答）：

1. **深度学习交叉方向的小白如何入门？**

​	A：主要注重工程实践和理论的结合，我的深度学习理论可能从前到后就学了两三个月，零零碎碎的时间。按我的经历而言，我会喜欢通过文献直接入门（参考：B站李沐深度学习中的论文解读专栏）+代码实战（参考：kaggle官网的教程，简短凝练，只是使用TensorFlow框架写的），上述阶段结束就可以试试做一些简单的项目，比如手写数字图像识别，或者垃圾邮件分类，具体看你更喜欢CV还是NLP还是XXX。这些资源完全可以在GITHUB、CSDN、知乎上找到，如果大家对这个有疑问我可以详细展开。

​	关于科研的入门，最好就是完成上述工作后，直接去找一个老师带，在工程中学，千万不要单独拿时间出来补充理论。

2. **好发文章的方向/老师**

​	A：这种问题太过宽泛，我一下子都能列举几十个方向，这有点为难我了。好发文章也分为一遍期刊文章、会议文章（CCF-A/B）、子刊文章和Nature、Science文章。对于一般期刊文章（比如SCI1-2区）的话，直接追2024年之后才出现的热点，找项目复现，结合交叉领域改造创新即可。对于会议文章，大致也是如此，但需要有非A+B+C模块化的设计思路，能够有novelty，让人觉得interesting，找一个没有人考虑到的切入点很重要。对于子刊文章，除了具备较强的创新性，更多的是你要如何证明你的方法是开拓性且非常、非常、非常行之有效的，补充材料100页+已经司空见惯，纯方法论的提出是不可行的。

对于找老师，下次可以单独开一个小视频讲一下，这个还是非常有门道的。

3. **关于交叉的同学疑问：多学几个技能，还是专心一个技能？**

​	A：我个人的建议是多学几个技能，因为其实很多领域相互之间会有一些方法论的共鸣，这种共鸣很大程度会提升你的上限。早早的关注一个领域只会走入死胡同，哪怕已经成为了一个领域专精的人，因为眼界太短，很难再发现有价值的研究。而且本科期间没有必要定死一个方向，如果这个方向不喜欢、未来被迫改方向、这个方向因为时代原因走向与AI交叉转行，你之前的努力都将白费。

4. **自学材料/网课推荐：**

​	A：这个主题会在我们的计划之中，敬请期待！
